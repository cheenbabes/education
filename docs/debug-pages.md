# Debug & Development Pages

These pages are **not linked in the nav** but remain accessible by URL for development and testing.

## Pages

| Route | Purpose |
|---|---|
| `/archetype-debug` | View all 72 primary+secondary archetype combinations with shareable text |
| `/matching-debug` | Curriculum matching algorithm debugger — select archetype, set preferences, see full scoring breakdown |
| `/personas` | Persona simulator — run predefined or custom personas through the quiz to validate archetype outcomes |
| `/explore/glyph-lab` | Visual glyph pack comparison for star map constellation/curriculum icons |
| `/curriculum-review` | Static HTML dump of the full curriculum database with philosophy scores |

## Query Params

| Route | Param | Effect |
|---|---|---|
| `/compass/quiz` | `?debug=true` | Shows archetype scoring breakdown panel during quiz |
